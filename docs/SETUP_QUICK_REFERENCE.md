## For a New Machine (5 Minutes)

### 1. Clone Repo
```powershell
git clone https://github.com/GarayInvestments/HouseRenoAI.git
cd HouseRenoAI
```

### 2. Run Automated Setup
```powershell
.\scripts\setup\setup-env.ps1
```

### 3. Add Secrets
Copy these files from secure backup:
- `.env` ‚Üí Project root
- `house-renovators-credentials.json` ‚Üí `config/`

### 4. Start Development
```powershell
# Terminal 1: Backend
.\.venv\Scripts\Activate.ps1
python -m uvicorn app.main:app --reload

# Terminal 2: Frontend
cd frontend
npm install
npm run dev
```

## What to Backup Before Switching Machines

### Required Files (Keep Secure!)
```
.env                                    # All environment variables
config/house-renovators-credentials.json  # Google Service Account
```

### Backup Methods (Choose One)

**Option A: Password Manager** ‚≠ê Recommended
- Store in 1Password/BitWarden
- Easy to retrieve on new machine
- Encrypted and synced

**Option B: Encrypted USB**
- Create encrypted zip with password
- Transfer physically
- Most secure for sensitive data

**Option C: Secure Cloud**
- Use encrypted cloud (ProtonDrive, Tresorit)
- Delete after transfer
- Good for remote setups

## Environment Variables You Need

### Must Have:
```env
SHEET_ID=your_sheet_id                # From Google Sheets URL
OPENAI_API_KEY=sk-proj-xxx            # From OpenAI dashboard
JWT_SECRET_KEY=random_32_char_string  # Auto-generated by setup script
```

### For QuickBooks Features:
```env
QUICKBOOKS_CLIENT_ID=xxx
QUICKBOOKS_CLIENT_SECRET=xxx
QUICKBOOKS_REDIRECT_URI=http://localhost:8000/v1/quickbooks/callback
```

### File Path:
```env
GOOGLE_SERVICE_ACCOUNT_FILE=config/house-renovators-credentials.json
```

## Common Issues

**"GOOGLE_SERVICE_ACCOUNT_FILE not found"**
‚Üí Place JSON in `config/` folder

**"Invalid API key"**
‚Üí Check `.env` for typos, no extra spaces

**"Module not found"**
‚Üí Activate venv: `.\.venv\Scripts\Activate.ps1`

**"Port 8000 already in use"**
‚Üí Kill old process or use `--port 8001`

## Getting Credentials

| Credential | Where to Get It |
|------------|----------------|
| Google Service Account | [Google Cloud Console](https://console.cloud.google.com) ‚Üí IAM ‚Üí Service Accounts |
